Canvas gradijent

Gradijent se koristi za popunjavanje unutrašnjosti pravougaonika, krugova, linija, tekstalnih sadržaja i slično.

Postoje dva tipa gradijenta:

  • createLinearGradient(x,y,x1,y1) - kreira linearni gradijent
  • createRadialGradient(x,y,r,x1,y1,r1) - kreira radijalni/kružni gradijent

Jednom gradijentu možemo dodati dve ili više boja.

Metoda addColorStop() određuje boju kojom se popunjava neki objekat i poziciju te boje duž gradijenta. Ta pozicija se kreće u opsegu između 0 i 1.

Da bismo koristili gradijent, podesimo fillStyle ili strokeStyle property na definisani gradijent. Nakon toga crtamo oblike.

Primer primene linearnog gradijenta nad pravougaonikom:

<canvas id="myCanvasL" width="200" height="100" style="border:1px solid #c3c3c3;"></canvas> <script> var c = document.getElementById("myCanvasL"); var ctx = c.getContext("2d"); var grd = ctx.createLinearGradient(0, 0, 200, 0); grd.addColorStop(0, "red"); grd.addColorStop(1, "white"); ctx.fillStyle = grd; ctx.fillRect(10, 10, 150, 80); </script>

Primer primene radijalnog gradijenta nad pravougaonikom:

<canvas id="myCanvasR" width="200" height="100" style="border:1px solid #c3c3c3;"></canvas> <script> var c = document.getElementById("myCanvasR"); var ctx = c.getContext("2d"); var grd = ctx.createRadialGradient(75, 50, 5, 90, 60, 100); grd.addColorStop(0, "red"); grd.addColorStop(1, "white"); ctx.fillStyle = grd; ctx.fillRect(10, 10, 150, 80); </script>
