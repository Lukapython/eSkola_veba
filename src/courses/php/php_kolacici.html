Kolačići

Kolačić je mali fajl koji server ugrađuje u računar korisnika. Kolačiće koristimo često da bismo identifikovali korisnika. Postoje 3 koraka uključena u identifikaciju korisnika.
  • Server šalje kolačiće brauzeru. Npr. ime, godiste ili id.
  • Brauzer ove informacije skladišti u računar korisnika.
  • Sledeći put kad brauzer pošalje zahtev serveru, onda salje i te kolacice, koje server koristi da identifikuje korisnika.

Kreiranje kolačića

Za kreiranje kolačića koristimo funkciju setcookie() Sintaksa: setcookie(name, value, expire, path, domain, secure, httponly);
Funkcija setcookie mora da se pojavi pre etikete <html>
  • name - Ovo je jedini obavezan argument. Njime zadajemo ime kolačića. Njega koristimo za pristup kolačiću.
  • value - Ovim argumentom zadajemo vrednost globalnoj promenljivoj $_COOKIE['name'].
  • expiry - Predstavlja vreme od ponoći 1 Januara 1970. Nakon tog vremena više nećemo moći da pristupimo kolačiću
  • path - Precizira koji direktorijumi za koje je kolačić ispravan. Jedan karakter "/" omogućuje da kolacic bude ispravan za sve direktorijume.
  • domain - precizira domen na kome kolačić važi
  • security - podrazumevana vrednost je 0, a ako je postavimo na 1, tada se kolačić šalje preko HTTPS veze, a ne preko obične HTTP nezastićene veze
Primer: <?php setcookie("ime","Marko Markovic",time()+3600,"/","",0); setcookie("godine","36","time()+3600"); ?>

Pristup kolačiću

Najlaksi način za pristup kolačiću je pomoću globalne promenljive $_COOKIE. U narednom primeru pristupamo kolačiću iz prethodnog primera.
Primer: <!DOCTYPE html> <html> <head> </head> <body> <?php echo $_COOKIE['ime']."<br>"; echo $_COOKIE['godine']."<br>"; ?> </body> </html> Korisno je koristiti funkciju isset() da proverimo da li je vrednost kolačića postavljena ili nije. <!DOCTYPE html> <html> <head> </head> <body> <?php if(isset($_COOKIE['ime'])){ echo $_COOKIE['ime']."<br>"; } else{ echo "Nazalost, nije postavljena vrednost"; } ?> </body> </html>

Brisanje kolačića

Da obrišemo kolačić koristimo takođe funkciju setcookie, samo sa negativnim danom isteka.
Primer: <!DOCTYPE html> <html> <head> </head> <body> <?php setcookie("godine","",time()-3600); echo "Kolačić 'godine' je obrisan"; ?> </body> </html> Zadatak:
Dvema promenljiva ime i prezime dodeliti vrednosti. Zatim njihove vrednosti pomoću kolačića preneti na drugu stranicu, i ispisati ih.
