Ugrađene funkcije

Ugrađene funkcije su one koje ne moramo definisati prilikom izrade HTML dokumenta, vec ih možemo koristiti uvek kada nam zatrebaju.
Postoji mnoštvo ugrađenih funkcija. U nastavku su navedene neke od njih, one koje imaju široku primenu i čestu upotrebu.

Funkcija alert

U nastavku je prikazan primer HTML dokumenta, koji sadrži funkciju alert: 
<html>
  <head>
    <title>Funkcija alert</title>
  </head>
  <body>
    <script>
    alert("JavaScript");
    </script>
  </body>
</html>

Pokretanjem ovog dokumenta aktiviraće se iskačući prozor sa porukom JavaScript.

Funkcija confirm

Prikazan je primer HTML dokumenta, koji sadrži funkciju confirm: 
<html>
  <head>
    <title>Funkcija confirm</title>
  </head>
  <body>
    <script>
    var izaberi=confirm("Prihvatate li uslove privatnosti?");
    </script>
  </body>
</html>

Pozivanjem ovog dokumenta aktiviraće se iskačući prozor sa porukom Prihvatate li uslove privatnosti?.

Funkcija prompt

Prikazan je primer HTML dokumenta, koji sadrži funkciju prompt: 
<html>
  <head>
    <title>Funkcija prompt</title>
  </head>
  <body>
    <script>
    var ime=prompt("Unesite svoje ime:");
    </script>
  </body>
</html>

Pokretanjem ovog dokumenta aktiviraće se iskačući prozor sa porukom Unesite svoje ime:. Iskačući prozor sadrži tekstualno polje u koje je omogućen unos podataka. Ukoliko korisnik odabere dugme "OK" funkcija prosleđuje vrednost koja je uneta u tekstualno polje, a ako je odabrano dugme "Cancel" povratna vrednost će biti "null". Konkretno, u navedenom primeru, povratna vrednost funkcije se upisuje u promenljivu ime.

Funkcija write

Prikazan je primer HTML dokumenta, koji sadrži funkciju write: 
<html>
  <head>
    <title>Funkcija write</title>
  </head>
  <body>
    <script>
    document.write("Dobar dan!");
    </script>
  </body>
</html>

Pokretanjem ovog dokumenta u internet pretraživač se upiše tekst Dobar dan!.

Funkcija Date

Prikazan je primer HTML dokumenta, koji sadrži funkciju Date: 
<html>
  <head>
    <title>Funkcija Date</title>
  </head>
  <body>
    <script>
    var datum=Date();
    alert(datum);
    </script>
  </body>
</html>

Funkcija Date vraća vrednost trenutnog datuma, vremena i vremenske zone područja sa kog se poziva.
U navedenom primeru ove vrednosti se upisuju u novu promenljivu datum, a zatim se funkcijom alert ispisuju u iskačući prozor.

Primeri upotrebe nekih metoda

Metode nad objektima tipa Date: 
<html>
  <head>
    <title>Datum</title>
  </head>
  <body>
    <h1>Datum</h1>
    <script>
     var datum = new Date();
     document.write("Ovo je trentno vreme: <br>");
     document.write(datum);
     document.write("<br>");
     var datum1 = new Date("July 21, 1983, 01:15:00");
     document.write("Ovaj datum smo sami podesili: <br>");
     document.write(datum1);
     document.write("<br>");
     datum.setDate(15);
     document.write("Nakon metode setDate(), danasnji datum je: <br>");
     document.write(datum);
     document.write("<br>");
     var d = datum1.getDay();
     document.write("Metoda getDay() vratila je: <br>");
     document.write(d + ", a to je redni broj dana u nedelji.");
     </script>
   </body>
</html>


Metode nad numeričkim objektima: 
<html>
  <head>
    <title>Brojevi</title>
  </head>
  <body>
    <h1>Brojevi</h1>
    <script>
    var broj= prompt("Unesite neki broj:");;
    document.write("Unet je broj: <br>"+broj+"<br>");
    kvBroj=Math.pow(broj,2);
    document.write("Kvadrat datog broja je: <br> "+kvBroj.toString()+" <br>");
    var stringBroj=broj.toString();
    var podString=stringBroj.substr(0,1);
    document.write("Prva cifra unetog broja je: <br>"+podString);
    </script>
  </body>
</html>

Rad nekih od ovih funkcija je predstavljen i u prostoru za vežbu.

Zadatak: Ispitaj koji je rezultat rada funkcije "parseInt()" ukoliko joj se prosledi argument:
a) "25"
b) "25.2"
