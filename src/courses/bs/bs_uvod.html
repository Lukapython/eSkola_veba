Uvod u Bootstrap


Šta je Bootstrap?

Bootstrap je front-end css frejmvork čija je osnovna namena kreiranje responzivnih veb stranica, tj. stranica koje mogu da prilagođavaju svoj izgled u zavisnosti od rezolucije uredjaja na kojem se prikazuiju. Bootstrap pruža veoma dobar temelj za njihovo kreiranje zbog svoje prilagodljivosti i nudi način da se uštedi vreme, a postigne zadovoljavajući kvalitet. Nalazi se u klasi najpopularnijih radnih okruženja za razvoj veb aplikacija. Bootstrap uvodi osnovne HTML i CSS dizajnerske šablone za tipografiju, forme, dugmad, tabele, navigaciju, slike i mnogo toga drugog. Razvili su ga Mark Otto i Jacob Thornton u Twitter-u i objavili ga kao proizvod otvorenog koda u avgustu 2011. na GitHub-u.

Zašto koristiti Bootstrap?

  • Jednostavan je za upotrebu. Svako sa osnovnim znanjem iz HTML-a i CSS-a može početi sa korišćenjem Bootstrap okruženja.
  • Odgovarajući CSS okruženja Bootstrap se prilagođava telefonima, tabletima i desktop računarima.
  • Kompatibilan je sa svim modernim pregledačima (Chrome, Firefox, Internet Explorer, Safari i Opera).

Kreiranje prve stranice

Da bi se kreirala stranica korišćenjem Bootstrap okruženja potrebno je ispuniti sledeće zahteve.
  • Dodati <!DOCTYPE html> na početku strane.
  • Uključiti Bootstrap okruženje.
    • Za potrebe ovog kursa, Bootstrap okruženje biće uključeno preko CDN-a. Dakle, u prethodni kod se uključuju pomenute tri linije koda.
  • Navođenje <meta> etikete sa odgovarajućim svojstvima.
    • Potrebno je dodati sledeću liniju :
    <meta name="viewport" content="width=device-width, initial-scale=1"> unutar <head> elementa.
Ako se uzmu u obzir sva tri uslova, kod izgleda ovako: Da bi se napravile jednostavne stranice u okruženju Bootstrap, koriste se klase omotača. Postoje dve klase omotača :
  • Klasa .container obezbeđuje odgovarajući omotač fiksne širine.
  • Klasa .container-fluid obezbeđuje omotač u punoj širini.
Napomena: Omotač se ne može smestiti unutar nekog drugog omotača.
Primer u editoru koji se nalazi desno ilustruje upotrebu klase .container koja se dodeljuje jednom <div> elementu. Kod ovog primera je zapravo nadogradnja prethodnog koda. Ubačeno je sve što je potrebno da bi Bootstrap funkcionisao i dodat jedan <div> element.
Zadatak: Izmeni program tako da se koristi druga klasa omotača koja je pomenuta.
