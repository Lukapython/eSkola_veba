Zahtevi i odgovori


U nekoj od prethodnih lekcija prikazan je najjednostavniji oblik zahteva i odgovora i to sledećim kodom u web.php fajlu: Route::get('/', function () { echo "Hello World"; }); Sada će biti napravljen TestController koji izgleda ovako: <?php namespace App\Http\Controllers; use Illuminate\Http\Request; use Illuminate\Routing\Controller; class TestController extends Controller { // protected $request; public function testRequest(Request $request){ $this->request = $request; $uri = $request->path(); $url = $request->url(); var_dump($url); var_dump($uri); } } ?> U web.php fajl dodaćemo deo koda za rutiranje: Route::get('test', 'TestController@testRequest'); Ako se pokrene aplikacija sa http://localhost:8000/test, dobiće se izlaz string(26) "http://127.0.0.1:8000/test" string(4) "test". Ako se pokrene sa http://localhost:8000/test/hagudu/class=5/school=Don%20Bosco, odgovor će biti string(60) "http://localhost:8000/test/hagudu/class=5/school=Don%20Bosco" string(38) "test/hagudu/class=5/school=Don%20Bosco". Naravno, pre pokretanja aplikacije u web.php fajlu treba dodati Route::get('test/{name}/{class}/{school}', 'TestController@testRequest'); Ako treba prikazati url putanju, u kontroler treba da dodati $url = $request->url(); . Odgovor zahteva biće pun naziv putanje. Sledi prikaz nekoliko metoda koje mogu biti korisne ako je zadata forma. $name = $request->input('name'); $input = $request->only('username', 'password'); $input = $request->except('credit_card'); Preko rutiranja i kontrolera šalje se zahtev, a onda se dobija odgovor. U prethodnim lekcijama je prikazano da odgovor može biti tekst ako se navede Route::get('first', function () { return "Hello World" }); u web.php fajlu. Takođe, odgovor može biti view koji se nalazi u folderu resources. Poseban vid odgovora može se dobiti redirekcijom, što će biti obrađeno u sledećoj lekciji.
