Tipovi podataka, rad sa tabelama i kolonama


U ovoj lekciji će detaljnije biti opisani tipovi podataka u kolonama i neke korisne metode za rad sa tabelama i kolonama.

Tipovi podataka


Schema::create('primer', function($table) { $table->increments('id'); }); Ovaj kod se piše kada određenu kolonu treba postaviti da automatski povećava vrednost i da bude primarni ključ. U konkretnom primeru id je naziv kolone. Podaci koje će kolona sadržati mogu biti tipa string. Istom logikom, lako je podesiti kolonu da prihvata podatke tipa string, ali je moguće u kodu zadati kolika će biti maksimalna dužina stringa i to na sledeći način: Schema::create('primer', function($table) { $table->string('prezime', 150); }); Tipovi podataka mogu biti integer, boolean, string, enum, date, time... Sintaksa za definisanje kolone tipa enum je sledeća: Schema::create('primer', function($table) { $table->enum('who', ['Pera', 'Mika', 'Zika']); }); Sada će biti prikazane još neke korisne metode za podešavanje podataka kolona. Prva od njih je metoda unique(). Kada se navede ova metoda, ne dozvoljva se ponavljanje vrednosti. Na primer: Schema::create('primer', function($table) { $table->string('prezime')->unique(); }); Ovom metodom se u konkretnom primeru ne dozvoljava unos istog prezimena. Ako se želi naglasiti da će ta kolona biti i primarni ključ onda se može dodati jedan red više na sledeći način: Schema::create('primer', function($table) { $table->string('prezime')->unique(); $table->primary('username'); }); Sve ovo se može uraditi u jednom redu na sledeći način: Schema::create('primer', function($table) { $table->string('username')->unique()->primary(); }); Ako je potrebno postaviti vrednost nove kolone na podrazumevanu vrednost, to se može uraditi na sledeći način: Schema::create('primer', function($table) { $table->string('name')->default('admin'); });

Metode za promenu imena i uklanjanje tabela i kolona

Jedna od metoda je metoda za preimenovanje tabele, to je metoda rename() koja prima dva argumenta. Prvi argument je trenutni naziv tabele, a drugi argument je budući naziv tabele. Kod koji treba dopisati nakon tipova podataka je: Schema::rename('users', 'names'); Metod dropColumn() koristi se kada treba ukloniti kolonu iz tabele i to na sledeći način: Schema::table('primer', function($table) { $table->dropColumn('name'); Za uklanjanje tabele koristi se metod drop() koji kao argument prihvata naziv tabele. Schema::drop('primer'); Argument koji ovaj metod prima je naziv kolone koju treba ukloniti. Navođenjem više naziva može se ukloniti više kolona. Schema::table('primer', function($table) { $table->dropColumn('name', 'godine'); Metod renameColumn() služi za preimenovanje kolona.
