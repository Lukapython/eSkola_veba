Četvorougao <rect>

Etiketa <rect> se koristi kako bi se kreirao četvorougao i njegove varijacije.
<svg width = "400" height = "110"> <rect width = "300" height = "100" style = "fill : rgb(0,0,255); stroke-width : 3; stroke : pink; fill-opacity : 0.5; stroke-opacity : 0.9" /> </svg>
Prethodni kod daje sledeću sliku:

Objašnjenje koda:
  • Atributi width i height etikete <rect> definišu širinu i visinu četvorougla
  • Atribut style se koristi da definiše CSS svojstva za četvorougao
  • CSS svojstvo fill definiše boju unutrašnjosti četvorougla
  • CSS svojstvo stroke-width definiše širinu linije kojom se obeležava stranica četvorougla
  • CSS svojstvo stroke definiše boju linije kojom se obeležava stranica četvorougla
  • CSS svojstvo fill-opacity definiše providnost boje unutrašnjosti četvorougla (u opsegu: od 0 do 1), ukoliko se ne navede, podrazumevano je 1
  • CSS svojstvo stroke-opacity definiše providnost boje linije kojom se obeležava stranica četvorougla (u opsegu: od 0 do 1), ukoliko se ne navede, podrazumevano je 1

Za postavljanje providnosti cele slike, može se koristiti CSS svojstvo opacity (u opsegu od 0 do 1).
Kreiranje četvorougla sa zaobljenim uglovima: <svg width = "400" height = "180"> <rect x = "50" y = "20" rx = "20" y = "20" width = "150" height = "150" style = "fill : red; stroke : black; stroke-width : 5; opacity : 0.5" /> </svg>
Prethodni kod daje sledeću sliku:

Objašnjenje koda:
  • Atribut x definiše levu poziciju četvorougla (npr. x = "50" smešta četvorougao 50 px od leve margine), ukoliko se ne navede,
    podrazumevno je x = "0"
  • Atribut y definiše gornju poziciju četvorougla (npr. y = "20" smešta četvorougao 20 px od gornje margine), ukoliko se ne navede,
    podrazumevno je y = "0"
  • Atributi rx i ry zaobljavaju uglove četvorougla
    • Zadatak: Kreirati kvadrat čija je unutrašnjost crne boje, stranice zelene boje širine 10, a providnost 0.7.
