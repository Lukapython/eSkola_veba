Osobine teksta

Atributi fonta

Najosnovniji atributi koji odredjuju izgled slova su: font-family, font-size, font-weight, font-style, font-variant, font.
  • font-family - Izbor fonta za ispis teksta.
  • font-size - Veličina slova.
  • font-weight - Podebljani ispis slova.
  • font-style - Zakošen ispis slova.
  • font - Atribut za postavljanje više osobina odjednom.

font-family

Ovaj atribut odredjuje izgled slova kojim se ispisuje tekst. Moguće je navesti bilo koji font koji postoji na računaru, a neke od mogucnosti su:
  • serif - Font sa postoljima, podrazumevani je Times new roman.
  • sans-serif - Font bez postolja, podrazumevani je Arial.
  • monospace - Slova fiksne širine, podrazumevani je Courier New.
  • cursive - Kurzivni (iskošeni) font.
Ako želimo da zadamo sam naziv fonta koji u sebi ima razmak, pišemo ga unutar navodnika.
.primer1{ font-family: "Times New Roman", serif; }

font-size

Ovaj atribut određuje veličinu slova. Zadaje se kao:
  • numerička vrednost - Postavlja veličinu slova na određenu veličinu u pikselima.
  • procentualna vrednost - Postavlja veličinu slova na procenat u odnosu na matične elemente.
  • jedna od sledećih mogućnosti - xx-small, x-small, small, medium, large, x-large, xx-large.
.primer2{ font-size: 14px; font-size: 40%; }

font-weight

Ovaj atribut odredjuje debljinu slova kojim se tekst ispisuje.Unapred definisane vrednosti su:
  • normal - Obična slova.
  • bold - Podebljana slova.
  • bolder - Ukoliko je moguće, jače podebljana slova.
  • lighter - Slova tanja od normalnih.
.primer3{ font-weight: bold; }

font-style

Atribut koji odredjuje da li su slova iskošena.
  • normal - Obična slova.
  • italic - Iskošena slova.
.primer4{ font-style: italic; }

font

Atributom font mozemo istovremeno da postavimo više podešavanja za font slova. Redosled zadatih vrednosti jeste bitan.
  • Prvo se zadaju vrednosti za font-style i za font-weight.
  • Zatim se zadaje font-size.
  • I uvek se na kraju zadaje font-family.
.primer5{ font: bold, 14pt, "Times New Roman",serif; }
Primer
Sada ćemo na dva načina zadati potpuno iste parametre:
Prvo ćemo ih zadati pomoću pojedinačnih atributa:
.primer6{ font-family: "Times New Roman",serif; font-size: 11pt; font-weight: bold; font-style: italic; }
A zatim ćemo iste parametre zadati preko kompleksnog atributa font.
.primer7{ font: italic, bold, 11pt, "Times New Roman",serif; }
U ovom primeru slova su ispisana "Times New Roman" fontom,iskošena su,boldirana i njihova veličina je 11pt.

Atributi teksta

Atributi teksta obuhvataju podešavanja vezana za font i pasus i to su:
  • text-decoration - Vrsta podvlačenja teksta.
  • text-transform - Transformacija u velika ili mala slova.
  • text-align - Poravnanje teksta.

text-decoration

Načini podvlačenja teksta su:
  • none - Slova bez linije.
  • underline - Linija se nalazi ispod teksta.
  • overline - Linija se nalazi iznad teksta.
  • line-through - Linija se iscrtava preko testa.
.primer8{ text-decoration: underline; }

text-transform

  • none - Slova bez promene.
  • capitalize - Svako prvo slovo reči postaje veliko.
  • uppercase - Sva slova pretvara u velika.
  • lowercase - Sva slova pretvara u mala.
.primer9{ text-transform: lowercase; }

text-align

Zadaje se za blok unutar kog se nalazi tekst. Poravnanje se vrši horizontalno prema ivicama bloka.
  • right - Tekst poravnat po desnoj strani.
  • left - Tekst poravnat po levoj strani.
  • center - Tekst je centriran.
  • justify - Tekst poravnat po obe ivice bloka.
.primer10{ text-align: center; }

Zadatak: Napisati pasus čiji će tekst biti centriran, veličina slova 12px, a slova da budu iskošena i boldirana.
