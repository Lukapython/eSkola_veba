Pseudo klase


Psedo klase su usko vezane za selektore i određuju stanje ili vezu prema selektoru. Imaju sledeću formu: selektor:pseudo_klasa{osobina:vrednost}

Linkovi


Linkovi, bilo posećeni (viseted) ili neposećeni link su najjednostavnije pseudo klase. Naime, menjaće se boje svih linkova na stranice u zavisnosti da li je korisnik posetio tu stranicu ili ne.

a:link { color: blue; } a:visited { color: purple; }

Dinamičke pseudo klase


Najčešće korišćene za linkove, dinamičke pseudo klase mogu se koristiti za primenu stilova u navedenim situacijama:
  1. active kada je nešto aktivirano od strane korisnika (recimo kada se klikne na link)
  2. hover kada jos uvek nije neka akcija preduzeta, već nam se pruza kao mogućnost (recimo prelazak preko linka)
  3. focus kada je nešto u fokusu, kada se selektuje ili se sprema za unos sa tastature

a:active { color: red; } a:hover { text-decoration: none; color: blue; background-color: yellow; } input:focus, textarea:focus { background: #eee; }
focus nije podržan od strane starijih pretraživača (browsers), stoga pažljivo!

First children


Konačno, tu je i pseudo klasa first child (prvo dete). Ovo se odnosi samo na prve potomke elemenata (glave, tela...)
<body> <p>I am the body&#8217;s first paragraph child. I rule. Obey me.</p> <p>I resent you.</p> ...
...ako želimo da stilizujemo sledeći paragraf, to radimo uz pomoć CSS koda:
p:first-child { font-weight: bold; font-size: 40px; }
