Animacije


Animacije omogućavaju da dati element promeni stil u neki drugi. Možemo promeniti koliko god svojstava želimo neograničen broj puta.
    Potrebno je obratiti pažnju na sledeće:
  • animation-name – određuje na koje CSS svojstvo ćemo primeniti prelaz ( inicijalna vrednost je all)
  • @keyframes pravilo – opisuje putanju promena koje vršimo na elementu
  • animation-duration - trajanje animacije; ako ga ne navedemo animacija se neće izvršiti jer je inicijalna vrednost 0s
  • animation-iteration-count -određuje koliko puta želimo da se animacija ponovi; vrednost infinitive će animaciju izvršavati beskonačan broj

Naredne linije koda prikazuju animaciju u kojoj element zelene boje prelazi u plavu u trajanju od 3s. Prvi primer sa desne strane izvršava bas ovu animaciju. 
@keyframes primer {
    from {background-color: green;}
    to {background-color: blue;}
}
div {
    width: 200px;
    height: 200px;
    animation-name: primer;
    animation-duration: 3s;
}


Možemo definisati @keyframes koji ima više promena/prelaza uz pomoć procenta na sledeći način:

@keyframes example {
    0%   {background-color: blue;}
    25%  {background-color: yellow;}
    50%  {background-color: red;}
    100% {background-color: green;}
}

    
Kao i u prethodnoj lekciji svojstvo animation-delay opisuje kašnjenje odnosno kada će prelaz početi da se izvršava.


Napomena:
U kodu sa desne strane u drugom primeru je korišćena funkcija animation-iteration-count sa vrednošću beskonačno (infinitive), pa se animacija stalno ponavlja, dok se prvi primer izvršava samo jednom. Primetiti da se nakon izvršenja animacije, objekat vraća na početno stanje. 



Zadatak:

Izmeniti kod u drugom primeru tako da menja pozicije kada menja i boju, tako da:

Kada je boja plava, pozicija 0px,0px.

Kada je boja zuta, pozicija 100px,100px.

Kada je boja crvena, pozicija 400px,0px.

Kada je boja zelena, pozicija 0px,0px.
