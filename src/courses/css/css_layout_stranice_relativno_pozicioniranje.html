Relativno pozicioniranje


Relativno pozicioniranje pomera element u odnosu na original, kao što je vec pomenuto, prostor koji bi element zauzimao pre pomeranja ostaje sačuvan i utiče na izgled stranice.
Moguce je preklapanje sa drugim elementima.
Zadatak: U primeru je sivi pravougaonik pozicioniran relativno. Primećuješ li prazninu koju bi ispunjavao da nije pozicioniran relativno? Postavi njegovu poziciju na static, šta uočavaš? Odakle merimo top i left?
