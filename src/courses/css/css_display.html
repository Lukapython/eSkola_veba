Display


Svojstvo display je jedno od najvažnijih svojstava CSS-a za kontrolisanje rasporeda elemenata na stranici koje određuje kako će se element prikazati. Vrednosti ovog svojstva su: block, inline i none.
Svaki HTML element ima određenu display vrednost u zavisnosti od toga kog je tipa, najčešće je to blok(block) ili linijski(inline).
Primeri blok elemenata:
<div> <h1> - <h6> <p> <form> <header> <footer> <section>
Primeri linijskih elemenata:
<span> <a> <img>
Korišćenjem display svojstva može se neki linijski element prikazati kao blok, a i obrnuto.
Svojstvo none se koristi ako ne želimo da se neki element prikaže.

Svojstvo visibility sa vrednošću hidden je slično kao i display: none, ali postoji razlika. Display: none izbacuje u potpunosti element i oslobađa se prostor koji je pregledač rezervisao za njega, dok visibility: hidden zadržava rezervisano mesto koje je element imao i ne prikazuje njegov sadržaj.

Vrednost none je vrlo korisno koristiti za dinamičke efekte.
Inline prikazuje elemente redom jedan za drugim i to linijski. Ako listu koja je blok element prikažemo inline, njene stavke će se prikazati jedna za drugom, a ne jedna ispod druge, i na taj način se lista pretvara u horizontalnu navigaciju.
Sa druge strane, vrednost block prikazuje element samostalno, na primer box elementu postavlja prazan red iznad i ispod i omogućava manipulisanje visinom, marginama, padding.

Postoji i kombinacija, display: inline-block. Element će zadržati vrednost inline, dozvoljavaće da se linijski elementi pozicioniraju pored njega, ali će pozajmiti veću fleksibilnost formatiranja koju ima block, na primer biće moguće podešavanje širine, visine.

Zadatak: Pogledaj u primeru koja svojstva imaju paragrafi. Pokušaj da primenom display svojstva izbrišeš četvrti paragraf i dodaš šesti koji će biti pored petog.
