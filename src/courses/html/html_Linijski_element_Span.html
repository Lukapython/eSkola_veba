Linijski element <span>


Linijski elementi se nastavljaju ispisivati u istom redu. Ne mogu unutar sebe sadržati blok elemente, ali mogu linijske. Linijski elementi su <img>, <a>, <span>,...
    Tri osnovna svojstva <span> elementa:
  • style - definisanje CSS svojstva.
  • class - određuje naziv klase za pristup elementu iz CSS-a ili JavaScripta.
  • id - jedinstveni identifikator za pristup elementu iz CSS-a ili JavaScripta.
Primer: <blockquote style="font-size:larger>":Ako ti neko kaže: <span style="color:green;">"Ne možeš to",</span> odgovori mu: <span style="color:red;">"Mogu"</span>.</blockquote>
U ovom primeru smo koristili atribut style.
Zadatak: Promeni zelenu boju teksta iz primera.
