Interfejs SortedMap


Interfejs SortedMap proširuje interfejs Map i održava raspored stavki po rastucem redosledu kljuceva.
SortedMap je genericki interfejs koji se deklariše na sledeci nacin: Interface SortedMap<K,V> gde K oznacava tip kluceva, a V tip vrednosti koje ce mapa sadržati.
Metode koje definiše intefejs SortedMap prikazane su u tabeli 15.

Tabela 15. Metode interfejsa SortedMap
Metod Opis
Comparator Comparator ( ) Vraca komparator uredene mape. Ako je mapa uredena prema prirodnom redosledu, vraca vrednost null.
K first key ( ) Vraca prvi kljuc uredene mape.
SortedMap headMap (K kraj) Vraca uredenu podmapu stavki ciji su kljucevi manji od argumenta kraj.
K lastKey( ) Vraca poslednji kljuc mape.
SortedMap subMap (K pocetak, K kraj) Vraca mapu stavki s kljucevima koji pripadaju opsegu izmedu vrednosti pocetak I kraj -1.
SortedMap tailMap (K pocetak) Vraca podmapu stavki s kljucevima vecim od argumenta pocetak -1.

Uredene mape omogucavaju veoma efikasan rad sa podmapama. Podskupovi mape mogu se dobiti primenom metoda headMap(), tailMap() ili subMap(). Dobijene podmape sadrže reference na elemente izvorne mape, što znaci da menjanje elemenata podmape menja istovremeno odgovarajuci element mape, i obrnuto.
