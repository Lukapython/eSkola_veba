Interfejs SortedMap

Interfejs SortedMap proširuje interfejs Map i održava raspored stavki po rastućem redosledu ključeva.
SortedMap je generički interfejs koji se deklariše na sledeći način: Interface SortedMap<K,V> gde K označava tip klučeva, a V tip vrednosti koje ce mapa sadržati.
Metode koje definiše intefejs SortedMap prikazane su u tabeli 15.

Tabela 15. Metode interfejsa SortedMap
Metod Opis
Comparator <? Super K>Comparator ( ) Vraća komparator uređene mape. Ako je mapa uređena prema prirodnom redosledu, vraća vrednost null
K first key ( ) Vraća prvi ključ uređene mape.
SortedMap<K, V> headMap (K kraj) Vraća uređenu podmapu stavki čiji su ključevi manji od argumenta kraj.
K lastKey( ) Vraća poslednji ključ mape.
SortedMap<K, V> subMap (K početak, K kraj) Vraća mapu stavki s ključevima koji pripadaju opsegu između vrednosti početak I kraj -1.
SortedMap<K, V> tailMap (K početak) Vraća podmapu stavki s ključevima većim od argumenta početak -1.

Urđene mape omogućavaju veoma efikasan rad sa podmapama. Podskupovi mape mogu se dobiti primenom metoda headMap(), tailMap() ili subMap(). Dobijene podmape sadrže reference na elemente izvorne mape, što znači da menjanje elemenata podmape menja istovremeno odgovarajući element mape, i obrnuto.
