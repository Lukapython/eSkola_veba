Interfejs Map


Mape (eng. Maps) predstavljaju objekat u kojem se cuvaju veze izmedu parova kljuc/vrednost, pri cemu kljuc mora biti jedinstven, a vrednost se može duplirati. Mape su strukture podataka koje se koriste za efikasno nalaženja nekog elementa upotrebom kljuca.
Interfejs Map povezuje jedinstvene kljuceve sa vrednostima. Map je genericki interfejs koji se deklariše na sledeci nacin: Interface Map<K,V> gde K oznacava tip kluceva, a V tip vrednosti koje ce mapa sadržati.
Metode koje definiše intefejs Map prikazane su u tabeli 14.

Tabela 14. Metode interfejsa Map
Metod Opis
void clear() Uklanja sve parove kljuc/ vrednost iz mape.
boolean containsKey(Object kljuc) Vraca true ako mapa sadrži kljuc, u suprotnom vraca vrednost false.
boolean containsValue(Object vrednost) Vraca true ako mapa sadrži vrednost, u suprotnom vraca vrednost false.
Set>entrySet() Vraca objekat tipa Set koji sadrži stavke mape.Taj skup se sastoji od objekata tipa MapEntry. To znaci da ova metoda omogucava prikaz sadržaja mape u obliku skupa.
boolean equals(Object obj) Vraca true ako mapa sadrži iste stavke kao i mapa obj. U suprotnom vraca false.
V get (Object kljuc) Vraca vrednost koja odgovara kljucu zadatom parametrom kljuc. Ako ne pronade kljuc, vraca null.
int hascode() Vraca heš-kljuc za pozivajucu mapu.
boolean isEmpty() Vraca true ako je mapa prazna, u suprotnom vraca vrednost false.
SetkeySet() Vraca objekat tipa Set koji sadrži kljuceve mape. Ova metoda omogucava prikaz kljuceva mape u obliku skupa.
V put(K kljuc, V vrednost) Unosi novu stavku u mapu, pri cemu nova vrednost zamenjuje postojecu vrednost koja odgovara zadatom kljucu. Ako zadati kljuc ne postoji, metoda vraca null. Ako kljuc postoji vraca prethodnu vrednost koja je s njim povezana.
void putAll(Mapmapa) Unosi u proizvoljnu mapu sve vrednosti iz mape.
V remove(Object kljuc) Uklanja stavku ciji je kljuc zadat parametrom kljuc.
int size() Vraca broj parova kljuc/vrednost u mapi.
Collection values() Vraca kolekciju koja se sastoji od vrednosti iz mape. Ova metoda omogucava prikazivanje sadržaja mape u obliku kolekcije.

Dve najvažnije metode ovog interfejsa su get() i put(). Metoda put() umece par kljuc/vrednost u mapu, dok metoda get() ucitava vrednost iz mape koja odgovara zadatom kljucu. Takode, mape ne impelmenitaju interfejs Collection, pa nisu kolekcije, ali postoje metode kojima možete sadržaj mape preslikati u kolekciju. Tome služe metode entrySet(), keySet() i values().
