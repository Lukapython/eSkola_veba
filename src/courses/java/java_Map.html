Interfejs Map

Mape (eng. Maps) predstavljaju objekat u kojem se čuvaju veze između parova ključ/vrednost, pri čemu ključ mora biti jedinstven, a vrednost se može duplirati. Mape su strukture podataka koje se koriste za efikasno nalaženja nekog elementa upotrebom ključa.
Interfejs Map povezuje jedinstvene ključeve sa vrednostima. Map je generički interfejs koji se deklariše na sledeci način: Interface Map<K,V> gde K označava tip klučeva, a V tip vrednosti koje će mapa sadržati.
Metode koje definiše intefejs Map prikazane su u tabeli 14.

Tabela 14. Metode interfejsa Map
Metod Opis
void clear() Uklanja sve parove ključ/ vrednost iz mape.
boolean containsKey(Object kljuc) Vraća true ako mapa sadrži ključ, u suprotnom vraća vrednost false.
boolean containsValue(Object vrednost) Vraća true ako mapa sadrži vrednost, u suprotnom vraća vrednost false.
Set<Map.Entry<K,V>>entrySet() Vraća objekat tipa Set koji sadrži stavke mape. Taj skup se sastoji od objekata tipa MapEntry. To znači da ova metoda omogućava prikaz sadržaja mape u obliku skupa.
boolean equals(Object obj) Vraća true ako mapa sadrži iste stavke kao i mapa obj. U suprotnom vraća false.
V get (Object kljuc) Vraca vrednost koja odgovara ključu zadatom parametrom ključ. Ako ne pronađe kljuc, vraca null.
int hascode() Vraća heš-ključ za pozivajuću mapu.
boolean isEmpty() Vraća true ako je mapa prazna, u suprotnom vraća vrednost false.
Set<K>keySet() Vraća objekat tipa Set koji sadrži ključeve mape. Ova metoda omogućava prikaz ključeva mape u obliku skupa.
V put(K kljuc, V vrednost) Unosi novu stavku u mapu, pri čemu nova vrednost zamenjuje postojeću vrednost koja odgovara zadatom ključu. Ako zadati ključ ne postoji, metoda vraca null. Ako ključ postoji vraća prethodnu vrednost koja je s njim povezana.
void putAll(Map<?extends K,? extends V>mapa) Unosi u proizvoljnu mapu sve vrednosti iz mape.
V remove(Object kljuc) Uklanja stavku ciji je kljuc zadat parametrom kljuc.
int size() Vraća broj parova ključ/vrednost u mapi.
Collection <V> values() Vraća kolekciju koja se sastoji od vrednosti iz mape. Ova metoda omogućava prikazivanje sadržaja mape u obliku kolekcije.

Dve najvažnije metode ovog interfejsa su get() i put(). Metoda put() umeće par ključ/vrednost u mapu, dok metoda get() učitava vrednost iz mape koja odgovara zadatom kljuću. Takođe, mape ne impelmenitaju interfejs Collection, pa nisu kolekcije, ali postoje metode kojima možete sadržaj mape preslikati u kolekciju. Tome služe metode entrySet(), keySet() i values().
