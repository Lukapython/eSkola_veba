Klasa HashSet

Klasa HashSet pravi kolekciju čiji se elementi čuvaju u obliku heš-tabele. Heš-tabela čuva podatke koristeći heširanje (eng. hashing), U mehanizmu heširnja, na osnovu vrednosti zadatog ključa izračunava se jedinstvena vrednost, tzv. heš-kod (eng. hash-code). Heš-kod se zatim koristi kao indeks za smeštanje podataka koji su pridruženi ključu. Heš-kod se automatski izračunava na osnovu izvornog ključa. Prednost heširanja je to što vreme izvršavanja metoda add(), contains(), remove() i size() ostaje isto, čak i u slučaju velikih skupova. Kako postupak izračunavanja heš-koda ne omogućava formiranje uređenih skupova, skup HashSet ne garantuje nikakav redosled elemenata. Zato jedan iterator za skup HashSet obilazi elemente u nepredvidljivom redosledu, koji čak može biti potpuno drugačiji nakon dodavanja novog elementa skupa. Ovo naravno nije ograničenje, ukoliko redosled nabrajanja elemenata nije bitan za rešavanje problema.

Klasa HashSet je generička klasa koja se definiše na sledeći nacin:
Class HashSet <E> U ovoj deklaraciji E je tip objekata koji se skaldišti.
Klasa HashSet realizuje interfejse Collection, Iterable i Set, što znači da klasa HashSet može koristiti sve metode definisane ovim interfejsima.
Konstruktori koje sadži klasa HashSet prikazani su u tabeli 10.

Tabela 10. HashSet konstruktori
Konstruktor Opis
HashSet() Pravi prazna skup.
HashSet(Collection ?extends E kolekcija) Pravi skup i inicijalizuje elementima kolekcije.
HashSet(int kapacitet) Pravi skup sa zadatim kapacitetom.
HashSet(int kapacitet, float popunjenost) Pravi skup i inicijalizuje kapacitet i popunjenost skupa.

Da bi objekat tipa HashSet bio kreiran i popunjen, neophodno je prvo kreirati njegovu instancu pomoću konstruktora iz tabele 11, a zatim dodati elemente koristeći metodu add(). Poslednji konstruktor, HashSet(int kapacitet, float popunjenost), pravi skup i inicijalizuje kapacitet i popunjenost skupa. U ovom slučaju popunjenost skupa je vrednost izmedu 0.0 i 1.0. Šta ova vrednost predstavlja? Kada broj elemenata u skupu postane veci od kapaciteta skupa pomnoženog s njegovom popunjenošcu, skup tipa HashSet se proširuje. Ukoliko koeficijent popunjenosti nije zadat konstruktorom, koristi se podrazumevana vrednost – 0,75.

Primer 13: Upotreba klase HashSet
import java.util.*; public class PrimerHashSet { public static void main(String[] args) { // Kreiranje objekta tipa HashSet HashSet<String> set1 = new HashSet<>(); // Dodavanje objekata u set1 - ilustracija metoda add() set1.add("Beograd"); set1.add("Vranje"); set1.add("Nis"); set1.add("Cacak"); set1.add("Novi Sad"); System.out.println("Sadrzaj skupa1: " + set1); System.out.println("Broj elemenata u skupu:" + set1.size() ); // Brisanje objekata iz HashSet objekta - ilsutracija metoda remove() set1.remove("Nis"); System.out.println("\nSadrzaj skupa1 nakon brisanja objekta: " + set1); System.out.println("Broj elemenata u skupu:" + set1.size()); // Kreiranje objekta tipa HashSet HashSet<String> set2 = new HashSet<String>(); // Dodavanje objekata u set2 - ilustracija metoda add() set2.add("Pariz"); set2.add("London"); set2.add("Milano"); // Dodavanje elemenata u skupa2 skupu1 - Ilustracija metoda addAll() set1.addAll(set2); System.out.println("\nSadrzaj skupa1 nakon dodavanja elemenata iz skupa2: " + set1); // Brisanje svih elemenata iz skupa2 - ilustracija metoda remvoveAll() set1.removeAll(set2); System.out.println("Sadrzaj skupa1 nakon brisanja elemenata iz skupa2: " + set1); } }

Klasa HashSet implementira interfejs Set, što znaci da duplikati elemanata nisu dopušteni. Metoda add() u tom slucaju vraca vrednost false i element se ne dodaje.
Kao što je već naglašeno, kolekcije mogu da uskladište svaku vrstu objekata, pa i objekte klasa koje programer sam kreira. Dakle, i kolekcija tipa HashSet se može koristiti za skladištanje objekata klasa koje programer sam kreira.

Važne osobine HashSet klase:
  • HashSet kolekcija ne skaldišti objekte redosledom kojim se dodaju
  • HashSet kolekcija ne dozvoljava dupliranje elemenata
  • Obezbeđuje vrlo efikasne operacije nalaženja, dodavanja i brisanja elemenata
