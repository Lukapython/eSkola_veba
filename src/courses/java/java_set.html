Interfejs Set

Ovaj interfejs definiše skup elementa. Skup predstavlja kolekciju podataka kod koje ne postoje duplikati elemenata. Interfejs Set nasleđuje interfejs Collection i ne definiše nikakve dodatne metode. Dakle, interfejs Set sadrži samo metode definisane interfejsom Collection uz dodato ogrničenje o jedinstvenosti elemenata. To znači da pri pokušaju dodavanja elementa koji već postoji , metoda add() nasleđena iz interfejsa Collection vraća vrednost false.

Set je generički interfejs koji se deklariše na sledeći način:
Interface Set <E> gde E označava tip objekta koji će skup sadržati.
