Interfejs Set


Ovaj interfejs definiše skup elementa. Skup predstavlja kolekciju podataka kod koje ne postoje duplikati elemenata. Interfejs Set nasleduje interfejs Collection i ne definiše nikakve dodatne metode. Dakle, interfejs Set sadrži samo metode definisane interfejsom Collection uz dodato ogrnicenje o jedinstvenosti elemenata. To znaci da pri pokušaju dodavanja elementa koji vec postoji , metoda add() nasledena iz interfejsa Collection vraca vrednost false.

Set je genericki interfejs koji se deklariše na sledeci nacin:
Interface Set <E> gde E oznacava tip objekta koji ce skup sadržati.
