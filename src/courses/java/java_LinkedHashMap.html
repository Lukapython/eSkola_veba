Klasa LinkedHashMap

Klasa LinkedHashMap proširuje klasu HashMap. Ona održava povezanu listu stavki mape redosledom njihovog umetanja u mapu, što omogucava pristupanje elementima mape po istom redosledu.
Klasa LinkedHashMap je genericka klasa koji se deklariše na sledeci nacin: Class LinkedHashMap <K,V> gde K oznacava tip kluceva, a V tip vrednosti koje ce mapa sadržati.
Konstruktori koje sadži klasa LinkedHashMap prikazani su u tabeli 18.

Tabela 18. LinkedHashMap konstruktori
Kontruktor Opis
LinkedHashMap Pravi praznu heš-mapu.
LinkedHashMap(Mapmapa) Pravi heš-mapu i inicijalizije je vrednostima iz mape.
HashMap(int kapacitet) Pravi heš-mapu sa zadatim kapacitetom.
HashMap(int kapacitet, float popunjenost) Pravi heš-mapu i inicijalizuje kapacitet i popunjenost mape.
HashMap(int kapacitet, float popunjenost, boolean redosled) Pravi heš-mapu i inicijalizuje kapacitet i popunjenost mape.Omogucava da zadate da li ce se elementima mape pristupiti po redosledu umetanja, ili redosledu poslednjeg iscitavanja.

Da bi objekat tipa LinkedHashMap bio kreiran i popunjen, neophodno je prvo kreirati njegovu instancu pomocu konstruktora iz tabele 17, a zatim dodati elemente koristeci metod put(). Kao što je objašnjeno, popunjenost mape je vrednost izmedu 0.0 i 1.0. Ukoliko koeficijent popunjenosti nije zadat konstruktorom, koristi se podrazumevana vrednost – 0,75. Podrazumevani kapacitet je 16. Poslednji konstruktor, HashMap(int kapacitet, float popunjenost, boolean redosled), omogucava zadavanje redosleda pristupanja. Ukoliko parameter redosled ima vrednost true koristi se redosled poslednjeg pristupanja. Ukoliko parameter ima vrednost false, koristi se redosled umetanja.

Klasa LinkedHashMap dodaje jednu svoju metodu:
Protected boolean removeEldestEntry(Map.Entry<K,V>stavka)
Ovu metodu pozivaju metode put() i putAll(). Najstarija stavka mape prosleduje se parametrom stavka. Standardna metod vraca false i ne radi ništa, ali ukoliko se redefiniše ova metoda, može se postoci da LinkedHashMap ukloni najstariju stavku iz mape. Da bi se to postiglo neophodno je da redefinsiana metoda vrati true. Da bi se zadržala najstarija stavka mape, redefinisana metoda treba da vrati false.

Primer 24: Upotreba klase LinkedHashMap
import java.util.*; public class PrimerZaLinkedHashMap { public static void main(String[] args) { // Kreiranje objekata tipa LinkedHashMap LinkedHashMap<String, Double> linkedHashMap = new LinkedHashMap<String, Double>(); // dodavanje elemenata - ilustracija metoda put() linkedHashMap.put("Marija Kojic", new Double(9.89)); linkedHashMap.put("Petar Petrovic", new Double(6.8)); linkedHashMap.put("Ana Stajic", new Double(10.00)); linkedHashMap.put("Teodora Mancic", new Double(7.21)); linkedHashMap.put("Vojin Janjic", new Double(8.2)); // Prikazivanje sadrzaja linkedHashMape System.out.println("Sadrzaj LinkedHashMape: " + linkedHashMap); // Pronalazenje prosecne ocena za Anu Stajic - ilustracija metoda get() System.out.println("\nProsecna ocena Ane Stajic: " + linkedHashMap.get("Ana Stajic")); // Velicina LinkedHashMape - ilustracija metoda size() System.out.println("\nVelicina LinkedHashMape: " + linkedHashMap.size()); // Provera da li je linkedHashMapa prazna - ilustacij ametoda isEmpty() System.out.println("\nDa li je LinkedHashMapa prazna? : " + linkedHashMap.isEmpty()); //Provera da li linkedHashMapa sadrzi kljuc ili vrednost - ilustracija metoda containsKey(), containsValue() System.out.println("\nLinkedHashMap sadrzi Nevenu Stajic? : " + linkedHashMap.containsKey("Nevena Stajic")); System.out.println("LinkedHashMap sadrzi prosecnu ocenu 10.0? : " + linkedHashMap.containsValue(10.0)); // Brisanje objekta iz mape- ilustracija metoda remove() System.out.println("\nBrisanje Teodore Mancic iz mape: ") linkedHashMap.remove("Teodora Mancic"); System.out.println("Sadrzaj mape nakon brisanja elemenata " + linkedHashMap); // Brisanje mape - ilustracija metoda clear() linkedHashMap.clear(); System.out.println("Sadrzaj mape nakon brisanja sadrzaja: " + linkedHashMap); } }
