Razlika između ArrayList klase i LinkedList klase

ArrayList i LinkedList klase implementiraju interfejs List, odnosno omogućavaju slične operacija nad listom. Postavlja se pitanje, zašto su onda potrebne dve implementacije liste umesto samo jedne. Odgovor leži u efikasnosti pojedinih operacija. Dvostruko povezana lista predstavljena klasom LinkedList efikasnija je u primenama kod kojih se elementi često dodaju ili uklanjaju iz sredine liste. Kod ArrayList objekta, odnosno kod dinamičkog niza, ove operacije zahtevaju pomeranje približno polovine elemenata udesno ili ulevo kako bi se napravilo dovoljno mesta za dodavanje novog elementa, ili kako bi se popunila praznina nakon uklanjanja nekog elementa. Sa druge stane, lista predstavljena ArrayList klasom efikasnija je kada treba pročitati ili promeniti proizvoljan element liste. U tom slučaju se trženom elementu pristupa korišćenjem njegovog indeksa, dok se kod povezane liste moraju redom ispitati svi elementi od početka, do u najgorem slučaju kraja liste. Takođe, ove klase ne implemetiraju iste interfejse. Za razliku od ArrayList klase, LinkedList klasa implementira i interfejse Queue i Deque, pa je spisak metoda koje se mogu primeniti nad LinkedList objektom malo veći.
