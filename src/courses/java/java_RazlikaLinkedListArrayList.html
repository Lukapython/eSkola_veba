Razlika izmedu ArrayList klase I LinkedList klase

ArrayList i LinkedList klase implementiraju interfejs List, odnosno omogucavaju slicne operacija nad listom. Postavlja se pitanje, zašto su onda potrebne dve implementacije liste umesto samo jedne. Odgovor leži u efikasnosti pojedinih operacija. Dvostruko povezana lista predstavljena klasom LinkedList efikasnija je u primenama kod kojih se elementi cesto dodaju ili uklanjaju iz sredine liste. Kod ArrayList objekta, odnosno kod dinamickog niza, ove operacije zahtevaju pomeranje približno polovine elemenata udesno ili ulevo kako bi se napravilo dovoljeno mesta za dodavanje novog elementa, ili kako bi se popunila praznina nakon uklanjanja nekog elementa. Sa druge stane, lista predstavljena ArrayList klasom efikasnija je kada treba procitati ili promeniti proizvoljan element liste. U tom slucaju se trženom elementu pristupa korišcenjem njegovog indeksa, dok se kod povezane liste moraju redom ispitati svi elementi od pocetka, do u najgorem slucaju kraja liste. Takode, ove klase ne implemetiraju iste interfejse. Za razliku od ArrayList klase, LinkedList klasa implemenira i interfejse Queue i Deque, pa je spisak metoda koje se mogu primeniti nad LinkedList objektom malo veci.
