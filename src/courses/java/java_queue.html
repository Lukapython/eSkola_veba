Interfejs Queue

Red (eng. queue) je kolekcija u kojoj se elementi čuvaju pre obrade. Elementi se iz reda mogu dohvatiti ili ukloniti samo sa vrha reda. To znači, da nije moguće preskočiti početni element i dohvatiti ili ukloniti neki drugi element iz reda. Sa druge strane, dodavanje elementa moguće je samo na kraj reda, pod uslovom da za element ima dovoljno mesta u redu.

Queue je genericki interfejs koji se deklariše na sledeći nacin:
Interface Queue<E> gde E označava tip objekta koji će red sadržati.
Ovaj interfejs nasleđuje interfejs Collection i dodaje metode definisane u tabeli 6.

Tabela 6. Metode definisane interfesjom Queue
Metoda Opis

E element()

Vraća element koji se nalazi na vrhu reda, pri čemu se taj element ne uklanja. Ako je red na čekanju prazan, generiše se izuzetak NoSuchElementExeption

boolean offfer(E obj)

Pokušava da doda objekat obj u red i vraća true ukoliko je operacija uspela, u suprotnom, vraća false.

E peek()

Vraća element koji se nalazi na vrhu reda. Ako je red prazan, vraća null. Element se ne uklanja.

E poll()

Vraća element koji se nalazi na vrhu reda, uklanjajući pri tom taj element. Ako je red prazan, vraća null

E remove()

Uklanja element koji se nalazi na vrhu reda i pri tom ga vraća kao rezultat metode. Ako je red prazan, generiše se izuzetak NoSuchElementExeption

Postoje dve metode koje učitavaju i vade element iz reda – poll() i remove(). Razlika između ovih metoda je u tome što metoda poll() vraća null ako je red prazan, a metoda remove() generiše izuzetak NoSuchElementExeption. Takođe, postoje i dve metode koje učitavaju ali ne vade element iz reda – peek() i element(). Kao i kod prethodnih metoda, prva metoda vraća null ako je red prazan, dok druga metoda generiše izuzetak. Kako postoje redovi fiksne dužine koji se mogu napuniti, metoda offer() pokušava da doda element u red. Dakle, ukoliko je red pun, ova metoda će vratiti vrednost false.

Izuzeci koji se generišu prilikom rada sa ovim metodama su:
  • ClassCastExeption – Ovaj izuzetak se generiše kada su dva objekta medusobno nekompatibilna
  • IlegalArgumentExeption – Ovaj izuzetak se generiše ukoliko je metodi prosleden nedozvoljeni argument
  • NullPointerExeption – Ovaj izuzetak se generiše pri pokušaju smeštanja null objekata u red
  • NoSuchElementExeption – Ovaj izuzetak se generiše kada u redu nema elemenata
