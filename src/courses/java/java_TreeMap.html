Klasa TreeMap

Klasa TreeMap za smeštaj podataka koristi hijerarhijsko stablo. Ova klasa pruža efikasan nacin za cuvanje uredenih parova kljuc/vrednost, zbog cega omogucava brzo ucitavanje podataka. Za razliku od HashMape ova klasa garantuje da ce njeni elementi biti razvrstani prema rastucem redosledu kljuceva. Klasa TreeMap imlementira intefejs Map i SortedMap, što znaci da klasa TreeSet može koristiti sve metode definisane ovim interfejsima
Klasa TreeMap je genericka klasa koji se deklariše na sledeci nacin: Class TreeMap <K,V> gde K oznacava tip kluceva, a V tip vrednosti koje ce mapa sadržati.
Konstruktori koje sadži klasa TreeMap prikazani su u tabeli 18.

Tabela 17.TreeMap konstruktori
Kontruktor Opis
TreeMap() Pravi praznu mapu u obliku stabla koja je uredena po prirodnom redosledu kljuceva.
TreeMap(Mapmapa) Pravi mapu u obliku stable i inicijalizije je stavkama iz mape, koje ureduje prirodnim redosledom.
TreeMap(Comparator komparator>) Pravi praznu mapu u obliku stable uredenu pomocu komapratora.
TreeMap(SortedMap uredena mapa) Pravi mapu u obliku stable i I iniscjalizuje je vrednostima iz uredenemape, održavajuci isti redosled stavki.

Da bi objekat tipa TreeMap bio kreiran i popunjen, neophodno je prvo kreirati njegovu instancu pomocu konstruktora iz tabele 17, a zatim dodati elemente koristeci metodu put().

Primer 22: Upotreba klase TreeMap
import java.util.*; public class PrimerZaTreeMap { public static void main(String[] args) { //Pravljenje mape u oblikustabla - povezivanje imena osobe sa stanjem tekuceg racuna TreeMap <String, Double> tm = new TreeMap<String, Double>(); // Stavljanje elemenata u mapu - ilustracija metoda put() tm.put("Nevena Petrovic", new Double(5857.34)); tm.put("Maja Pavlovic", new Double(255.84)); tm.put("Vladimir Gojkovic", new Double(1378.37)); tm.put("Ana Kojic", new Double(99.32)); tm.put("Teodora Mancic", new Double(-585.34)); // Formiranje skupa stavki mape - ilustracija metoda entrySet() Set<Map.Entry<String, Double>> skup= tm.entrySet(); // Prikazivanje skupa - ilustracija metoda getKey() i getValue() for (Map.Entry<String, Double>element:skup ){ System.out.print(element.getKey() + ":"); System.out.println(element.getValue()); } System.out.println(); // Deponovanje 2000 na racun Maje Pavlovic - ilustracija metoda get() double stanje = tm.get("Maja Pavlovic"); tm.put("Maja Pavlovic", stanje + 2000); // Provera stanja na racunu Maje Pavlovic System.out.println("Novo stanje racuna Maje Pavlovic:"+ tm.get("Maja Pavlovic")); // Brisanje objekta iz mape - islutracija metoda remove() tm.remove("Ana Kojic"); System.out.println("Sadrzaj mape nakon brisanja Ane Kojic:"); System.out.println(tm); }
Rezultat programa:
Ana Kojic:99.32 Maja Pavlovic:255.84 Nevena Petrovic:5857.34 Teodora Mancic:-585.34 Vladimir Gojkovic:1378.37 Novo stanje racuna Maje Pavlovic:2255.84 Sadrzaj mape nakon brisanja Ane Kojic: {Maja Pavlovic=2255.84, Nevena Petrovic=5857.34, Teodora Mancic=-585.34, Vladimir Gojkovic=1378.37}
Dakle, klasa TreeMap ureduje kljuceve. U ovom slucaju oni nisu poredani po prezimenima vec po imenima. Ukoliko je potrebno urediti kljuceve po prezimenima, potrebno je da u trenutku pravljenja mape zadate komparator.

Primer 23: Upotreba komapratora za sortiranje racuna
import java.util.*; // Upotreba komparatora za sortiranje racuna prema prezimenu vlasnika public class MojKomarator implements Comparator<String> { // redefinisananje metoda comapre public int compare(String a, String b) { int i,j, k; String aStr=a; String bStr=b; // Pronalazenje indeksa pocetka prezimena i = aStr.lastIndexOf(' '); j= bStr.lastIndexOf(' '); k = aStr.substring(i).compareTo(bStr.substring(j)); // ako se prezimena poklapaju, poredenje imena i prezimena zajedno if (k==0) return aStr.compareTo(bStr); else return k; } } import java.util.Map; import java.util.Set; import java.util.TreeMap; public class TreeMapPrimer { public static void main(String[] args) { //Pravljenje mape u obliku stabla - povezivanje imena osobe sa stanjem tekuceg racuna TreeMap <String, Double> tm = new TreeMap<String, Double>(new MojKomarator()); // Stavljanje elemenata u mapu - ilustracija metoda put() tm.put("Nevena Petrovic", new Double(5857.34)); tm.put("Maja Pavlovic", new Double(255.84)); tm.put("Vladimir Gojkovic", new Double(1378.37)); tm.put("Ana Gojkovic", new Double(99.32)); tm.put("Teodora Mancic", new Double(-585.34)); // Formiranje skupa stavki mape - ilustracija metoda entrySet() Set<Map.Entry<String, Double>> skup= tm.entrySet(); // Prikazivanje skupa - ilustracija metoda getKey() i getValue() for (Map.Entry<String, Double>element:skup ){ System.out.print(element.getKey() + ":"); System.out.println(element.getValue()); } System.out.println(); // Deponovanje 2000 na racun Maje Pavlovic - ilustracija metoda get() double stanje = tm.get("Maja Pavlovic"); tm.put("Maja Pavlovic", stanje + 2000); // Provera stanja na racunu Maje Pavlovic System.out.println("Novo stanje racuna Maje Pavlovic:"+ tm.get("Maja Pavlovic")); // Brisanje objekta iz mape - islutracija metoda remove() tm.remove("Ana Kojic"); System.out.println("Sadrzaj mape nakon brisanja Ane Kojic:"); System.out.println(tm); }
Klasa MojKomparator poredi dva znakovna niza koji sadrže imena i prezimena. To radi tako što prvo poredi prezimena. Najpre traži indeks razmaka u svakom ulaznom znakovnom nizu, a zatim poredi podnizove svakog elementa koji pocinju tim indeksom. U slucaju da su prezimena ista, sortiranje se vrši po imenu. Zbog toga je u primeru, „Ana Gojkovic“ ispred „Vladimira Gojkovica“.
