Interfejs Map.Entry

Ovaj interfejs omogućava rad sa stavkama mape. Map.Entry je generički interfejs koji se deklariše na sledeći način: Interface Map.Entry<K,V> gde K oznacava tip klučeva, a V tip vrednosti koje će mapa sadržati.
Metode koje definiše intefejs Map.Entry prikazane su u tabeli 16.

Tabela 16. Metode interfejsa Map.Enrty
Metod Opis

boolean equals (Object obj)

Vraća true ako objekat obj koji implementira interfejs Map.Entry sadrži isti ključ I vrednost kao pozivajući objekat.

K getKey ( )

Vraća ključ svake mape.

V getValue ( )

Vraća vrednost svake mape.

int hashCode ( )

Vraća heš-ključ svake mape.

V setValue  (Object vrednost)

Stavki mape dodeljuje vrednost. Ako vrednost nije odgovarajućeg tipa, metoda generiše izuzetak ClassCastException. Ako ima drugih problema s vrednošcu generiše se izuzetak IllegalArgumentException. Ako je zadata vrednost null, a mapa to ne dozvoljava, generiše se izuzetak NullPointerException. Ako se mapa ne može menjati, generiše se izuzetak UnsupportedOperationException .
