Interfejs Map.Entry

Ovaj interfejs omogucava rad sa stavkama mape. Map.Entry je genericki interfejs koji se deklariše na sledeci nacin: Interface Map.Entry<K,V> gde K oznacava tip kluceva, a V tip vrednosti koje ce mapa sadržati.
Metode koje definiše intefejs Map.Entry prikazane su u tabeli 16.

Tabela 16. Metode interfejsa Map.Enrty
Metod Opis
boolean equals (Object obj) Vraca true ako objekat obj koji implementira interfejs Map.Entry sadrži isti kljuc I vrednost kao pozivajuci objekat.
K getKey ( ) Vraca kljuc svake mape.
V getValue ( ) Vraca vrednost svake mape.
int hashCode ( ) Vraca heš-kljuc svake mape.
V setValue (Object vrednost) Stavki mape dodeljuje vrednost. Ako vrednost nije odgovarajuceg tipa, metoda generiše izuzetak ClassCastException. Ako ima drugih problema s vrednošcu generiše se izuzetak IllegalArgumentException. Ako je zadata vrednost null, a mapa to ne dozvoljava, generiše se izuzetak NullPointerException. Ako se mapa ne može menjati, generiše se izuzetak UnsupportedOperationException.
