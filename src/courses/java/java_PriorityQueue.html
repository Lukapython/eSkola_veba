Klasa PriorityQueue

Klasa PriorityQueue definiše red čekanja čijim elementima prioritet određuje zadati komparator. Ako prilikom definisanja reda nije zadat komparator, koristi se podrazumevani komparator za tip podataka koji se smeštaju u red. Podrazumevani komparator slaže elemente reda rastućim poretkom. Tako će na početku reda biti najmanja vrednost. Ukoliko zadate svoj namenski komparator možete zadati drugačiji redosled čekanja u redu. Tipična upotreba reda sa prioritetom je za rasporđivanje poslova. Svaki posao ima prioritet. Poslovi se dodaju proizvoljnim redom. Kad god treba započeti novi posao, iz reda se uklanja posao najvišeg prioriteta.

Klasa PriorityQueue je generička klasa koja se definiše na sledeći način:
Class PriorityQueue <E> U ovoj deklaraciji E je tip objekata koji se skaldišti.
Klasa PriorityQueue realizuje interfejse Collection, Iterable i Queue, što znači da klasa PriorityQueue može koristiti sve metode definisane ovim interfejsima.
Konstruktori koje sadži klasa PriorityQueue prikazani su u tabeli 13.

Da bi objekat tipa PriorityQueue bio kreiran, neophodno je prvo kreirati njegovu instancu pomoću konstruktora iz tabele 13. U svim ovim slučajevima kapacitet se povećava kada se objekat tipa PriorityQueue popuni elementima.

Primer 18: Upotreba klase PriorityQueue
import java.util.PriorityQueue; public class PriorityQueuePrimer { public static void main(String[] args) { // Kreiranje objekata tipa PriorityQueue koji sadrzi objekte tipa Integer PriorityQueue<Integer> red = new PriorityQueue<Integer>(); // Dodavanje objekata tipa Integer u red – ilustracija metoda offer() red.offer(5); red.offer(8); red.offer(22); red.offer(1); red.offer(29); // Ispisivanje sadrzaja reda System.out.print("Sadrzaj reda: "); for(Integer element: red){ System.out.print(element + " "); } } }
Rezultat programa:
Sadrzaj reda: 1 5 22 8 29 U ovom primeru koršćen je podrazumevani komparator za objekte tipa Integer. Kao što je već naglašeno, moguće je kreirati i sopstveni komparator i na taj način zadati drugačiji redosled čekanja u redu.

this.zarada = zarada; } public String toString(){ return ime_i_prezime +" : "+zarada; } } import java.util.Comparator; public class KomparatorZaPriorityQueue implements Comparator<Zaposleni>{ // Ova klasa implementira interfejs Comparator i sortira zaposlene po zaradi // redefinsanje metoda compare - metoda poredi zaposlene po zaradi public int compare(Zaposleni z1, Zaposleni z2) { return z1.zarada - z2.zarada;
