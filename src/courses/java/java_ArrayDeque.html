Klasa ArrayDeque


Klasa ArrayDeque predstavlja dinimicki niz neogranicenog kapaciteta, odnosno red. Ovaj red funkcioniše kao standardni red (eng. queue) po principu ”prvi koji ude, prvi izlazi“ (eng. FIFO – first in – first out), ili kao stek, po principu „poslednji koji ude, prvi izlazi“ (eng LIFO – last in – last out). Dodavanje elemenata u sredinu nije podržano.

Klasa ArrayDeque je genericka klasa koja se definiše na sledeci nacin:
Class ArrayDeque <E> U ovoj deklaraciji E je tip objekata koji se skaldišti.
Klasa ArrayDeque realizuje interfejse Collection, Iterable i Deque, što znaci da klasa ArrayDeque može koristiti sve metode definisane ovim interfejsima.
Konstruktori koje sadži klasa ArrayDeque prikazani su u tabeli 14.

ArrayDeque(Collection kolekcija)) Pravi red cekanja koji se inicijalizuj elementima kolekcije prosledene parametrom kolekcija.

Da biste bi objekat tipa ArrayDeque bio kreiran, neophodno je prvo kreirati njegovu instancu pomocu konstruktora iz tabele 13. U svim navedenim slucajevima, kapacitet ArrayDeque objekata se povecava kada se red popuni elementima.

Primer 20:Upotreba ArrayDeque klase
import java.util.*; public class PrimerZaArrayDeque { public static void main(String[] args) { // Kreiranje objekta tipa ArrayDeque ArrayDeque<String> adg = new ArrayDeque<String>(); // Objekat tipa ArrayDeque koristimo kao stek – ilustracija metoda push() adg.push("A"); adg.push("B"); adg.push("F"); adg.push("G"); System.out.print("Praznjenje steka: "); // Praznjenje steka - ilustracija metoda peek() i pop() while(adg.peek()!= null){ System.out.print(adg.pop() + " "); System.out.print(" "); } } }
Rezultat programa:
Praznjenje steka: G F B A U ovom primeru klasa ArrayDeque je upotrebljena za izradu steka. Kao što je prikazano u rezultatu programa, element koj je poslednji ušao u red, prvi je izašao iz njega. U ovom primeru korišcenje su metode Deque instefejsa.


Važne osobine ArrayDeque klase:
  1. Za razliku od obicnog reda, moguce je dodavati i brisati elemente sa oba kraja reda u kolekciji tipa ArrayDeque
  2. ArrayDeque je red neogranicenog kapaciteta.
  3. ArrayDeque je brzi od Steka i LinkedList kolekcije.
  4. Null objekti su dopušteni.
