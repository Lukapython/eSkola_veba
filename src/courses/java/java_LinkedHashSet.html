Klasa LinkedHashSet

Za razliku od HashSet klase koja ne garantuje redosled skladištenja objekata, klasa LinkedHashSet održava povezanu listu elemenata skupa koje skladišti redosledom kojim se umeću. Iz navedenog razloga, moguće je pristupiti elementima ovakvog skupa pomoću iteratora, a da pritom oni budu nabrajani redosledom kojim su uneti.

Klasa LinkedHashSet je generička klasa koja se definiše na sledeci način:
Class LinkedHashSet <E> U ovoj deklaraciji E je tip objekata koji se skaldišti.
Klasa LinkedHashSet realizuje interfejse Collection, Iterable i Set, što znači da klasa LinkedHashSet može koristiti sve metode definisane ovim interfejsima.

Konstruktori koje sadži klasa LinkedHashSet prikazani su u tabeli 11.
Tabela 11. LinkedHashSet konstruktori
Konstruktor Opis
LinkedHashSet() Pravi prazan skup.
LinkedHashSet(Collection ?extends E kolekcija) Pravi skup i inicijalizuje elementima kolekcije.
LinkedHashSet(int kapacitet) Pravi skup sa zadatim kapacitetom.
HashSet(int kapacitet, float popunjenost) Pravi skup i inicijalizuje kapacitet i popunjenost skupa.

Da bi objekat tipa LinkedHashSet bio kreiran i popunjen, neophodno je prvo kreirati njegovu instancu pomoću konstruktora iz tabele 11, a zatim dodati elemente koristeči metodu add().

Primer 14: Upotreba klase LinkedHashSet
import java.util.*; class LinkedHashSetPrimer { public static void main(String[] args) { // Kreiranje LinkedHashSet objekta LinkedHashSet<String> set = new LinkedHashSet<>(); // Dodavanje stringova u skup – ilisutracija metoda add() set.add("London"); set.add("Sarajevo"); set.add("Vranje"); set.add("Trst"); set.add("Beograd"); set.add("New York"); System.out.println(set); // Ispis elemenata malim slovima for (Object element: set) System.out.print(((String) element).toLowerCase() + ","); } }
Rezultat programa:
[London, Sarajevo, Vranje, Trst, Beograd, New York] london,sarajevo,vranje,trst,beograd,new york Klasa LinkedHashSet implementira interfejs Set, što znači da duplikati elemanata nisu dopušteni. Metoda add() u tom slučaju vraća vrednost false, i element se ne dodaje. Kolekcija tipa LinkedHashSet može se takođe koristiti za skladištanje objekata klase koje programer sam kreira.

Važne osobine LinkedHashSet klase:
  • LinkedHashSet kolekcija skaldišti objekte redosledom kojim se dodaju
  • LinkedHashSet kolekcija ne dozvoljava dupliranje elemenata
  • Iterator uvek obilazi elemente redosledom kojim su elementi dodati u kolekciju
