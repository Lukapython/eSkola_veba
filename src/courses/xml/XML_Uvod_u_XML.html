Uvod u XML

XML (skraćeno od eXtensible Markup Language) je meta jezik za obeležavanje. Koristi se za opisivanje, skladištenje i prenošenje podataka. Preporučen je 1998. godine od strane W3C (World Wide Web Consortium) iz potrebe za postojanjem jezika za obeležavanje, jednostavnim za parsiranje i obradu. Može se reći da XML predstavlja informacije o informaciji, upakovane u tagove. Danas nije dovoljno imati samo informaciju, već je potrebno da infromacije imaju opis ili obeležje, kako bi se mogle lakše identifikovati i koristiti, a XML nam upravo to omogućava.

Iako liči na HTML, nije osmišljen kao zamena za njega. Zapravo, XML se suštinski razlikuje od HTML-a. HTML je namenjen prikazu podataka, i od velikog je značaja kako prikazani podaci izgledaju, dok XML ima fokus na samim podacima, odnosno njihovom smeštanju i opisivanju, a ne sadrži informacije o prikazu.

Neke prednosti XML-a:
  • jednostavan i jasan za korišćenje ("self-descriptive")
  • pojednostavljuje prenos, razmenu i dostupnost podataka
  • podaci se smeštaju isključivo u tekstualnom formatu što omogucava skladištenje i prenos podataka nezavisno od platforme, odnosno hardvera i softvera koji se koristi
  • proširiv je i fleksibilan, trpi dodavanje i brisanje podataka
  • ima format koji je čitljiv za ljude, računare, glasovne mašine, itd.
