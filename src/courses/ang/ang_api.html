AngularJS API



API je skraćenica od Application Programming Interface.

AngularJS Global API



AngularJS Global API je skup globalnih JavaScript funkcija za obavljanje uobičajenih zadataka kao što su:
  • Upoređivanje objekata
  • Ponavljanje objekata
  • Pretvaranje podataka


Global API funkcijama se pristupa pomoću angular objekata.

Ispod je lista uobicajenih API funkcija:

API Opis
angular.lowercase() Prebacuje string u mala slova
angular.uppercase() Prebacuje string u velika slova
angular.isString() Vraća true ako je referenca string
angular.isNumber() Vraća true ako je referenca broj

Primeri:
  • angular.lowercase()

    • <!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <body> <div ng-app="myApp" ng-controller="myCtrl"> <p>{{ x1 }}</p> <p>{{ x2 }}</p> </div> <script> var app = angular.module('myApp', []); app.controller('myCtrl', function($scope) { $scope.x1 = "PETAR"; $scope.x2 = angular.lowercase($scope.x1); }); </script> </body> </html>

  • angular.uppercase()

    • <!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <body> <div ng-app="myApp" ng-controller="myCtrl"> <p>{{ x1 }}</p> <p>{{ x2 }}</p> </div> <script> var app = angular.module('myApp', []); app.controller('myCtrl', function($scope) { $scope.x1 = "Petar"; $scope.x2 = angular.uppercase($scope.x1); }); </script> </body> </html>

  • angular.isString()

    • <!DOCTYPE html> <html> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <body> <div ng-app="myApp" ng-controller="myCtrl"> <p>{{ x1 }}</p> <p>{{ x2 }}</p> </div> <script> var app = angular.module('myApp', []); app.controller('myCtrl', function($scope) { $scope.x1 = "PETAR"; $scope.x2 = angular.isString($scope.x1); }); </script> </body> </html>

  • angular.isNumber()

    Probajte sami:
    Napisati program koji, u zadatom stringu, mala slova prebacuje u velika,a velika u mala.
