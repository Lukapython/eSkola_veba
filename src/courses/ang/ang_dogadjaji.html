AngularJS događaji (events)

AngularJS ima sopstvene direktive za HTML događaje.

AngularJS događaji

Možete dodati osluškivače dogaćaja u vaše HTML elemente koristeći neku od sledećih direktiva:
  • ng-blur
  • ng-change
  • ng-click
  • ng-copy
  • ng-cut
  • ng-dblclick
  • ng-focus
  • ng-keydown
  • ng-keypress
  • ng-keyup
  • ng-mousedown
  • ng-mouseenter
  • ng-mouseleave
  • ng-mousemove
  • ng-mouseover
  • ng-mouseup
  • ng-paste
Direktive događaja nam dozvoljavaju da pokrećemo AngularJS funckije kod određenih korisničkih događaja.
AngularJS događaj neće ukloniti postojeći HTML događaj, oba će biti izvršena.

Događaji za mišem(Mouse Events)

Oni se dešavaju kad se kursor pomeri preko nekog elementa, po ovom redosledu:
  1. ng-mouseenter
  2. ng-mouseover
  3. ng-mousemove
  4. ng-mouseleave
Ili, kada se dugme miša klkne na odgovarajući element, u narednom redosledu:
  1. ng-mousedown
  2. ng-mouseup
  3. ng-click
Možete dodati događaj miša u bilo koji HTML element.
<div ng-app="mojaAplikacija" ng-controller="mojKontroler"> <h1 ng-mousemove="zbir = zbir + 1">Miš je preko mene!</h1> <h2>{{ zbir }}</h2> </div> <script> var app = angular.module('mojaAplikacija', []); app.controller('mojKontroler', function($scope) { $scope.zbir = 0; }); </script>

ng-click direktiva

Direktiva ng-click definiđe AngularJS kod koji će biti izvršen kada je kliknuto na element. <div ng-app="mojaAplikacija" ng-controller="mojKontroler"> <button ng-click="zbir = zbir + 1">Klikni me!</button> <p>{{ zbir }}</p> </div> <script> var app = angular.module('mojaAplikacija', []); app.controller('mojKontroler', function($scope) { $scope.zbir = 0; }); </script> Takođe možete i da referišete na funkciju: <div ng-app="mojaAplikacija" ng-controller="mojKontroler"> <button ng-click="mojaFunkcija()">Klikni me!</button> <p>{{ zbir }}</p> </div> <script> var app = angular.module('mojaAplikacija', []); app.controller('mojKontroler', function($scope) { $scope.count = 0; $scope.mojaFunkcija = function() { $scope.zbir++; } }); </script>

Toggle, True/False

Ako želite da prikažete deo HTML koda kada je dugme stisnuto i sakrijete kada je ponovo kliknuto, kao padajući meni, možemo napraviti da se dugme ponaša kao toggle prekidač: <div ng-app="mojaAplikacija" ng-controller="mojKontroler"> <button ng-click="mojaFunkcija()">Klikni me!</button> <div ng-show="pokaziMe"> <h1>Menu:</h1> <div>Piza</div> <div>Pasta</div> <div>Pesto</div> </div> </div> <script> var app = angular.module('mojaAplikacija', []); app.controller('mojKontroler', function($scope) { $scope.pokaziMe = false; $scope.mojaFunkcija = function() { $scope.pokaziMe = !$scope.pokaziMe; } }); </script> Promenjiva pokaziMe u početku ima Boolean vrednost false.
Funkcija mojaFunkcija postavlja vrdnost pokaziMe promenjive suprotno od onog što jeste, koristeći operator ! (not).

$event objekat

Možete da prosledite $event objekat kao argumetn kada pozivate funkciju.
$event objekat sadrži event objekat pregledača:
<div ng-app="mojaAplikacija" ng-controller="mojKontroler"> <h1 ng-mousemove="mojaFunkcija($event)">Miš je preko mene!</h1> <p>Koordinate: {{x + ', ' + y}}</p> </div> <script> var app = angular.module('mojaAplikacija', []); app.controller('mojKontroler', function($scope) { $scope.mojaFunkcija = function(myE) { $scope.x = myE.clientX; $scope.y = myE.clientY; } }); </script>
