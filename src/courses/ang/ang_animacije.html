AngularJS animacije



AngularJS pruža animirane prelaze, uz pomoć CSS.

Šta je animacija?



Animacija je kada transformacija HTML elementa daje iluziju kretanja.
Primer:
Označite polje za potvrdu da sakrijete odeljak:
<!DOCTYPE html> <html> <style> div { transition: all linear 0.5s; background-color: lightblue; height: 100px; width: 100%; position: relative; top: 0; left: 0; } .ng-hide { height: 0; width: 0; background-color: transparent; top:-200px; left: 200px; } </style> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-animate.js"></script> <body ng-app="ngAnimate"> <h1>Sakriti odeljak: <input type="checkbox" ng-model="myCheck"></h1> <div ng-hide="myCheck"></div> </body> </html>

Aplikacije ne bi trebalo da budu ispunjene animacijama, ali neke animacije mogu da čine aplikacije lakše razumljivim.

Šta vam je potrebno?



Da bi vaše aplikacije bile spremne za animacije, morate uključiti AngularJS Animate biblioteku:
<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-animate.js"> </script>

Onda se mora odnositi na ngAnimate modul u vašoj aplikaciji:
<body ng-app="ngAnimate">

Ili ako vaša aplikacija ima ime, dodati ngAnimate kao zavistan modul u svojoj aplikaciji.
Primer:
<!DOCTYPE html> <html> <style> div { transition: all linear 0.5s; background-color: lightblue; height: 100px; width: 100%; position: relative; top: 0; left: 0; } .ng-hide { height: 0; width: 0; background-color: transparent; top:-200px; left: 200px; } </style> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-animate.js"></script> <body ng-app="myApp"> <h1>Sakriti odeljak: <input type="checkbox" ng-model="myCheck"></h1> <div ng-hide="myCheck"></div> <script> var app = angular.module('myApp', ['ngAnimate']); </script> </body> </html>

Šta ngAnimate radi?


NgAnimate modul dodaje i uklanja klase.

NgAnimate modul ne animira svoje HTML elemente, ali kada ngAnimate primeti određene događaje, kao što su prikazati ili sakriti neki HTML element, element dobija neke prethodno definisane klase koje se mogu koristiti da bi napravili animacije.

Direktive u AngularJS koji dodaju / uklanjaju klase su:

  • ng-show

  • ng-hide

  • ng-class

  • ng-view

  • ng-include

  • ng-repeat

  • ng-if

  • ng-switch


ng-show i ng-hide direktive dodaju ili uklanjaju ng-hide vrednost klase.

Druge direktive dodaju ng-enter vrednost klase kada uđu u DOM, i ng-leave atribut kada su uklonjeni iz DOM.

Direktiva ng-repeat takođe dodaje ng-move vrednost klase kada HTML element menja položaj.

Pored toga, tokom animacije, HTML element će imati niz vrednosti klase, koji će biti uklonjeni kada se animacija završi.
Na primer ng-hide direktiva će dodati ove vrednosti klase:

  • ng-animate

  • ng-hide-animate

  • ng-hide-add (Ako je element biće sakriven)

  • ng-hide-remove (Ako je element biće prikazan)

  • ng-hide-add-active (Ako je element biće sakriven)

  • ng-hide-remove-active (Ako je element biće prikazan)


Korišćenje CSS animacija



Možete koristiti CSS tranzicije ili CSS animacije da animirate HTML elemente. Ovaj članak će vam pokazati kako.

CSS tranzicije



CSS tranzicije vam omogućavaju da lako menjate vrednosti CSS osobina, sa jedne vrednosti na drugu, u datom trenutku.
Primer:
Kada odeljak elementa dobije .ng-hide klasu, tranzicija će trajati 0,5 sekundi, a visina se glatko menja od 100pk do 0.
<!DOCTYPE html> <html> <style> div { transition: all linear 0.5s; background-color: lightblue; height: 100px; } .ng-hide { height: 0; } </style> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-animate.js"></script> <body ng-app="myApp"> <h1>Sakriti odeljak: <input type="checkbox" ng-model="myCheck"></h1> <div ng-hide="myCheck"></div> <script> var app = angular.module('myApp', ['ngAnimate']); </script> </body> </html>

CSS Animacije



CSS animacije vam omogućavaju da lako menjate vrednosti CSS osobina, sa jedne vrednosti na drugu, u datom trenutku.
Primer:
Kada odeljak elementa dobije .ng-hide klasu, myChange animacija će raditi, što će glatko menjati visinu od 100pk do 0.
<!DOCTYPE html> <html> <style> @keyframes myChange { from { height: 100px; } to { height: 0; } } div { height: 100px; background-color: lightblue; } div.ng-hide { animation: 0.5s myChange; } </style> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular.min.js"></script> <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.8/angular-animate.js"></script> <body ng-app="ngAnimate"> <h1>Sakriti odeljak: <input type="checkbox" ng-model="myCheck"></h1> <div ng-hide="myCheck"> </div> </body> </html>

Napisati program koji prilikom označavanja polja za potvrdu, odeljak proširuje do sredine ekrana.
